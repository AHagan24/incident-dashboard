## Features

- Create and track incidents
- Real-time cross-client updates with Socket.IO
- MongoDB persistence
- Status, priority, and severity tracking
- Analytics instrumentation with PostHog
- Responsive operations-style dashboard UI

## Tech Stack

- Next.js
- TypeScript
- MongoDB + Mongoose
- Socket.IO
- PostHog
- Tailwind CSS
